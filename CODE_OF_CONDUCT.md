# 行为准则（Code of Conduct）

本项目遵循 Contributor Covenant 行为准则（v2.1）的精神：尊重、包容、合作。

## 我们的承诺

我们致力于营造一个无骚扰的协作环境。无论年龄、体型、残障、族裔、性别认同与表达、经验水平、国籍、个人外貌、种族、宗教或性取向，任何人都应受到尊重。

## 不当行为示例

- 人身攻击、歧视、骚扰或侮辱性言论
- 发布他人隐私信息（doxxing）
- 恶意刷屏、挑衅、煽动对立

## 处理与申诉

如遇到违反行为准则的情况，请通过 GitHub Issues 或私信维护者进行反馈。维护者有权采取包括但不限于：警告、删除内容、关闭讨论、限制参与等措施。
